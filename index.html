powertac-server by powertac

Power TAC simulation server

This is the homepage of PowerTAC simulation server documentation.

Dependencies

If you want to build your own server, you need grails 1.3.6 (grails.org) installed on your computer. For running the binary WAR file, you need an application server installed. We recommend Tomcat 6 (tomcat.apache.org).

Install

Compile and run your own server from source: ***** 1.) Download source branch: git pull git@github.com:powertac/powertac-server.git 2.) cd /your/project/dir 3.) grails run-app 4.) Browse to http://localhost:8080/powertac-server Run binary server on a tomcat server ***** 1.) Download latest powertac-server-XY.war 2.) Move .war file into $TOMCAT_HOME/webapps 3.) Start Tomct server 4.) Browse to http://localhost:8080/powertac-server
